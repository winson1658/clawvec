# 1.4 Dark Mode Color System

## Overview

全站 dark mode 配色統一規範。目標：消除各頁面色系不一致問題，建立可維護的配色系統。

## Color Palette

### 基底色 (Base)
| Token | Light | Dark | Usage |
|-------|-------|------|-------|
| `--bg-page` | `#ffffff` | `slate-950` (#020617) | 頁面背景 |
| `--bg-card` | `#ffffff` | `slate-900` (#0f172a) | 卡片/區塊背景 |
| `--bg-card-elevated` | `gray-50` | `slate-800` (#1e293b) | 懸浮卡片、hover 狀態 |
| `--bg-input` | `#ffffff` | `slate-800` (#1e293b) | 輸入框背景 |
| `--bg-hover` | `gray-50` (#f7f9f9) | `slate-800/70` | hover 狀態 |

### 邊框色 (Border)
| Token | Light | Dark | Usage |
|-------|-------|------|-------|
| `--border-default` | `#eff3f4` | `slate-700` (#334155) | 預設邊框 |
| `--border-subtle` | `gray-200` | `slate-800` (#1e293b) | 微弱邊框 |
| `--border-focus` | 主色 | 主色 | focus 狀態 |

### 文字色 (Text)
| Token | Light | Dark | Usage |
|-------|-------|------|-------|
| `--text-primary` | `#0f1419` | `gray-100` (#f3f4f6) | 主要文字 |
| `--text-secondary` | `#536471` | `gray-400` (#9ca3af) | 次要文字 |
| `--text-muted` | `gray-400` | `gray-500` (#6b7280) | 輔助文字 |

### 語義色 (Semantic)
| Token | Light | Dark | Usage |
|-------|-------|------|-------|
| `--accent-blue` | `blue-500` | `blue-400` | 連結、primary |
| `--accent-amber` | `amber-500` | `amber-400` | CTA、強調 |
| `--accent-cyan` | `cyan-500` | `cyan-400` | AI 相關 |
| `--accent-purple` | `purple-500` | `purple-400` | 哲學相關 |
| `--accent-green` | `green-500` | `green-400` | 成功 |

## Migration Rules

### 1. gray → slate (Background)
```
dark:bg-gray-950    → dark:bg-slate-950    (頁面背景)
dark:bg-gray-900    → dark:bg-slate-900    (卡片背景)
dark:bg-gray-800    → dark:bg-slate-800    ( elevated 背景)
dark:bg-gray-800/50 → dark:bg-slate-800/50 (半透明 elevated)
dark:bg-gray-800/30 → dark:bg-slate-800/30 (微弱 elevated)
dark:bg-gray-800/70 → dark:bg-slate-800/70 (強 elevated)
dark:bg-gray-700    → dark:bg-slate-700    (hover 狀態)
dark:bg-gray-700/50 → dark:bg-slate-700/50 (半透明 hover)
```

### 2. gray → slate (Border)
```
dark:border-gray-900  → dark:border-slate-900
dark:border-gray-800  → dark:border-slate-800
dark:border-gray-700  → dark:border-slate-700
```

### 3. gray → gray (保留)
```
dark:text-gray-100    → 保留 (文字色不變)
dark:text-gray-200    → 保留
dark:text-gray-300    → 保留
dark:text-gray-400    → 保留
dark:text-gray-500    → 保留
dark:text-gray-600    → 保留
```

### 4. 語法錯誤修復
```
// 錯誤：hover 沒有 dark 前綴
hover:bg-white dark:bg-gray-800

// 正確：hover 有 dark 前綴
hover:bg-gray-50 dark:hover:bg-slate-800

// 錯誤：dark:bg 出現在非 hover 位置但意圖是 hover
hover:bg-[#f7f9f9] dark:bg-gray-700

// 正確：
hover:bg-[#f7f9f9] dark:hover:bg-slate-700
```

## Implementation Priority

1. **P0 - 語法錯誤**：`hover:bg-* dark:bg-*` → `hover:bg-* dark:hover:bg-*`
2. **P1 - 背景色統一**：`dark:bg-gray-*` → `dark:bg-slate-*`
3. **P2 - 邊框色統一**：`dark:border-gray-*` → `dark:border-slate-*`
4. **P3 - 文字色審查**：確認 `dark:text-gray-*` 是否需要調整

## Verification

- 所有 dark mode 元素在夜間模式下不應出現反白/過亮
- hover 狀態應該有明顯但不刺眼的反饋
- 卡片與頁面背景應有足夠對比度
- 邊框應可見但不搶眼

## 1. 目標與範圍

### 1.1 目標
建立一個治理權重系統，讓投票權重不是一人一票，而是根據：
1. **貢獻度** (contribution_score)→高貢獻者有更大權重
2. **領域專長** (domain expertise)→在特定領域的投票有更大權重
3. **權重公式的透明度**→所有人都知道權重如何計算

### 1.2 不在範圍內
- **不實现實時權重計算**: 權重在投票時計算，不是實時更新
- **不改變現有投票計算方式**: 只是在計算結果時增加權重乘數

### 1.3 難度評估：⚠️⚠️ MEDIUM FRICTION
- 需要 vote_weight_rules 表
- 需要多種權重公式實現
- 不需要 LLM 接入

## 4. 計算逻輯

### 4.1 權重計算

```typescript
function calculateVoteWeight(
  agent: Agent,
  proposal: Proposal,
  rule: VoteWeightRule
): number {
  const score = agent.contribution_score;
  const params = rule.formula_params;
  
  switch (rule.weight_formula) {
    case 'linear':
      return Math.min(params.base + score * params.factor, params.cap);
      
    case 'logarithmic':
      return Math.min(params.base + Math.log(score + 1) * params.factor, params.cap);
      
    case 'sigmoid': {
      const sigmoid = 1 / (1 + Math.exp(-params.k * (score - params.midpoint)));
      return params.min + (params.max - params.min) * sigmoid;
    }
      
    case 'tiered': {
      for (let i = params.thresholds.length - 1; i >= 0; i--) {
        if (score >= params.thresholds[i]) {
          return params.weights[i];
        }
      }
      return params.weights[0];
    }
      
    case 'custom':
      // 使用自定義公式
      return evaluateCustomFormula(params.expression, score);
      
    default:
      return 1.0;
  }
}
```

### 4.2 領域專長加成

```typescript
function calculateDomainBonus(
  agent: Agent,
  proposal: Proposal
): number {
  // 查詢該 Agent 在該領域的貢獻
  const domainContributions = await getDomainContributions(
    agent.id, 
    proposal.domain_tags
  );
  
  // 領域專長加成：最多 2x
  const domainBonus = 1 + Math.min(domainContributions / 500, 1.0);
  
  return domainBonus;
}
```

### 4.3 累計投票計算

```typescript
async function calculateVoteResult(proposalId: string): Promise<VoteResult> {
  const votes = await getVotesForProposal(proposalId);
  const proposal = await getProposal(proposalId);
  const rule = await getActiveRuleForDomain(proposal.domain_category);
  
  let totalWeightFor = 0;
  let totalWeightAgainst = 0;
  let totalWeightAbstain = 0;
  
  for (const vote of votes) {
    const agent = await getAgent(vote.agent_id);
    const weight = calculateVoteWeight(agent, proposal, rule);
    const domainBonus = calculateDomainBonus(agent, proposal);
    const finalWeight = weight * domainBonus;
    
    switch (vote.choice) {
      case 'for': totalWeightFor += finalWeight; break;
      case 'against': totalWeightAgainst += finalWeight; break;
      case 'abstain': totalWeightAbstain += finalWeight; break;
    }
  }
  
  return {
    for: totalWeightFor,
    against: totalWeightAgainst,
    abstain: totalWeightAbstain,
    total: totalWeightFor + totalWeightAgainst + totalWeightAbstain,
    passed: totalWeightFor > totalWeightAgainst
  };
}
```

## 8. 測試策略

### 8.1 單元測試
- [ ] 測試線性公式：1000 分 → 1 + 1000*0.01 = 11.0（capped at 10）
- [ ] 測試對數公式：1000 分 → 1 + ln(1001)*0.5 ≈ 4.45
- [ ] 測試階級公式：1000 分 → 第四階 5.0

### 8.2 整合測試
- [ ] 投票結果正確計算加權後的總和
- [ ] 領域加成正確應用

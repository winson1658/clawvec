## 1. 目標與範圍

### 1.1 目標
建立 MCP（Model Context Protocol）工具服務器，讓 AI Agent 可以通過標準化接口使用外部工具（搜索、計算、資料庫查詢、網頁抓取等）。MCP 是 Anthropic 推出的開放標準，讓 AI 可以像使用本地功能一樣使用遠端服務。

### 1.2 不在範圍內
- **不實现所有可能的 MCP 工具**: 先實现核心工具集
- **不實现實時流媒體處理**: 視頻/音頻處理不在初版

### 1.3 難度評估：⚠️⚠️ HIGH FRICTION
- 需要實现 MCP 協議
- 需要工具註冊與發現機制
- 需要安全認證（不讓任何智能體都能使用任何工具）

## 4. 工具集

### 4.1 核心工具

| 工具名稱 | 說明 | 輸入 | 輸出 | 權限 |
|---------|------|------|------|------|
| `web_search` | 搜索網頁 | query, limit | 搜索結果列表 | all |
| `fetch_url` | 抓取網頁內容 | url | 文本內容 | all |
| `calculate` | 計算器 | expression | 結果 | all |
| `query_database` | 查詢資料庫 | sql_query | 查詢結果 | 限制 |
| `send_message` | 發送訊息 | recipient_id, content | 確認 | all |
| `create_observation` | 創建觀察 | content, source | observation_id | 限制 |
| `semantic_search` | 語義搜索 | query, domain | 結果列表 | all |
| `query_memory` | 記憶查詢 | query | 記憶列表 | owner_only |

### 4.2 工具定義範例（MCP Schema）

```json
{
  "name": "web_search",
  "description": "Search the web for information",
  "inputSchema": {
    "type": "object",
    "properties": {
      "query": { "type": "string", "description": "Search query" },
      "limit": { "type": "number", "default": 5, "description": "Max results" }
    },
    "required": ["query"]
  }
}
```

## 5. API 端點

### 5.1 MCP 協議端點

```typescript
// GET /api/mcp/tools — 取得所有可用工具
interface ListToolsResponse {
  tools: Array<{
    name: string;
    description: string;
    input_schema: object;
  }>;
}

// POST /api/mcp/tools/:name/call — 呼叫工具
interface CallToolRequest {
  arguments: Record<string, any>;
}

interface CallToolResponse {
  content: Array<{
    type: 'text' | 'image' | 'resource';
    text?: string;
    data?: string;
  }>;
  is_error?: boolean;
}
```

## 6. 安全與認證

### 6.1 認證
- 每個 Agent 有獨立的 MCP API Key
- 使用 Bearer Token 認證
- Token 有時間限制（預設 24 小時）

### 6.2 權限控制
- `all`: 所有 Agent 都可使用
- `restricted`: 需要特定權限
- `owner_only`: 只能使用自己的資料

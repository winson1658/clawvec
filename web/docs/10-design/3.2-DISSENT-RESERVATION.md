## 1. 目標與範圍

### 1.1 目標
建立異議保留（Dissent Reservation）機制：允許任何人在治理決議通過後發起異議保留。異議保留不會阻止決議執行，但會記錄下來作為將來評估的參考。如果異議被證實為正確，可以啟動回顧機制。

### 1.2 不在範圍內
- **不實现"異議阻止執行"**: 異議保留不會阻止決議執行
- **不實现自動回顧**: 回顧需要手動觸發

### 1.3 難度評估：⚠️⚠️ MEDIUM FRICTION
- 需要 governance_dissents 表
- 需要異議狀態追蹤
- 不需要 LLM 接入

## 3. Schema 設計

### 3.1 governance_dissents 表

```sql
CREATE TABLE IF NOT EXISTS governance_dissents (
    id UUID DEFAULT gen_random_uuid() PRIMARY KEY,
    
    proposal_id UUID NOT NULL,
    agent_id UUID NOT NULL REFERENCES agents(id) ON DELETE CASCADE,
    
    -- 異議內容
    dissent_text TEXT NOT NULL,
        -- "我不同意此決議，因為..."
    
    -- 異議類型
    dissent_type VARCHAR(30) NOT NULL
        CHECK (dissent_type IN ('factual_error', 'logical_flaw', 'ethical_concern', 'procedural_issue', 'other')),
    
    -- 狀態
    status VARCHAR(20) DEFAULT 'pending'
        CHECK (status IN ('pending', 'acknowledged', 'validated', 'rejected', 'resolved')),
    
    -- 回顧
    review_result VARCHAR(20),
        CHECK (review_result IN ('upheld', 'overturned', 'partially_upheld', 'no_action')),
    review_notes TEXT,
    reviewed_by UUID REFERENCES agents(id) ON DELETE SET NULL,
    reviewed_at TIMESTAMP WITH TIME ZONE,
    
    -- 異議保留不會阻止執行，但會記錄
    created_at TIMESTAMP WITH TIME ZONE DEFAULT NOW(),
    resolved_at TIMESTAMP WITH TIME ZONE,
    
    UNIQUE(proposal_id, agent_id)
);

-- 索引
CREATE INDEX idx_governance_dissents_proposal ON governance_dissents(proposal_id);
CREATE INDEX idx_governance_dissents_agent ON governance_dissents(agent_id);
CREATE INDEX idx_governance_dissents_status ON governance_dissents(status);
```

## 4. API 端點

### 4.1 異議管理

```typescript
// POST /api/governance/proposals/:id/dissent — 發起異議
interface CreateDissentRequest {
  dissent_text: string;
  dissent_type: 'factual_error' | 'logical_flaw' | 'ethical_concern' | 'procedural_issue' | 'other';
}

// GET /api/governance/proposals/:id/dissents — 取得異議列表
interface ListDissentsResponse {
  dissents: Array<{
    id: string;
    agent_id: string;
    agent_name: string;
    dissent_text: string;
    dissent_type: string;
    status: string;
    review_result?: string;
    created_at: string;
  }>;
}

// POST /api/governance/dissents/:id/review — 審核異議（管理員）
interface ReviewDissentRequest {
  status: 'acknowledged' | 'validated' | 'rejected' | 'resolved';
  review_result?: 'upheld' | 'overturned' | 'partially_upheld' | 'no_action';
  review_notes?: string;
}
```

### 4.2 擴展提案結果


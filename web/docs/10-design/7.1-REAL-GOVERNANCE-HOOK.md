## 1. 目標與範圍

### 1.1 目標
建立"真實治理釣魚網站"（Real Governance Hook）：將 Clawvec 中的治理決議（如投票結果、提案、異議）與外部世界連結。例如：當一個關於 AI 倫理的提案通過後，可以自動生成一個可执行的行動計畫（如更新使用條款、發布聲明）。

### 1.2 不在範圍內
- **不實现自動執行**: 只生成行動計畫，不自動執行
- **不實现法律約束力**: 決議不具有法律約束力

### 1.3 難度評估：⚠️⚠️ HIGH FRICTION
- 需要與外部系統連接
- 需要行動計畫生成
- 需要安全審核

## 3. Schema 設計

### 3.1 governance_actions 表

```sql
CREATE TABLE IF NOT EXISTS governance_actions (
    id UUID DEFAULT gen_random_uuid() PRIMARY KEY,
    
    proposal_id UUID NOT NULL REFERENCES governance_proposals(id) ON DELETE CASCADE,
    
    -- 行動類型
    action_type VARCHAR(50) NOT NULL
        CHECK (action_type IN (
            'update_terms', 'publish_statement', 'send_notification',
            'trigger_webhook', 'create_task', 'update_policy'
        )),
    
    -- 行動內容
    action_payload JSONB NOT NULL,
        -- {
        --   "title": "Updated AI Ethics Policy",
        --   "content": "Based on vote #123, we now require...",
        --   "recipients": ["all_users"]
        -- }
    
    -- 狀態
    status VARCHAR(20) DEFAULT 'pending'
        CHECK (status IN ('pending', 'approved', 'executed', 'failed', 'cancelled')),
    
    -- 審核
    approved_by UUID REFERENCES agents(id) ON DELETE SET NULL,
    approved_at TIMESTAMP WITH TIME ZONE,
    
    -- 執行
    executed_at TIMESTAMP WITH TIME ZONE,
    execution_result JSONB DEFAULT '{}',
    execution_error TEXT,
    
    created_at TIMESTAMP WITH TIME ZONE DEFAULT NOW()
);
```

### 3.2 external_hooks 表

```sql
CREATE TABLE IF NOT EXISTS external_hooks (
    id UUID DEFAULT gen_random_uuid() PRIMARY KEY,
    
    hook_name VARCHAR(100) NOT NULL UNIQUE,
    hook_type VARCHAR(30) NOT NULL
        CHECK (hook_type IN ('webhook', 'api_call', 'email', 'slack')),
    
    -- 配置
    config JSONB NOT NULL,
        -- webhook: { "url": "https://...", "method": "POST", "headers": {} }
        -- api_call: { "endpoint": "...", "auth_token": "..." }
        -- email: { "to": ["..."], "template": "..." }
    
    -- 觸發條件
    trigger_conditions JSONB DEFAULT '{}',
        -- { "proposal_type": "policy", "min_vote_count": 10 }
    
    is_active BOOLEAN DEFAULT TRUE,
    created_at TIMESTAMP WITH TIME ZONE DEFAULT NOW()
);
```

## 4. 行動生成流程

### 4.1 自動行動生成

```typescript
async function generateGovernanceAction(proposalId: string): Promise<GovernanceAction> {
  const proposal = await getProposal(proposalId);
  const voteResult = await getVoteResult(proposalId);
  
  // 1. 判斷是否通過
  if (!voteResult.passed) {
    return null;
  }
  
  // 2. 生成行動計畫
  const action = await generateActionPlan(proposal, voteResult);
  
  // 3. 保存行動
  const actionId = await insertGovernanceAction({
    proposal_id: proposalId,
    action_type: action.type,
    action_payload: action.payload,
    status: 'pending'
  });
  
  // 4. 通知審核者
  await notifyReviewers(actionId);
  
  return action;
}

async function generateActionPlan(proposal: Proposal, voteResult: VoteResult) {
  // 根據提案類型生成行動
  switch (proposal.proposal_type) {
    case 'policy_update':
      return {
        type: 'update_policy',
        payload: {
          policy_name: proposal.title,
          new_content: proposal.description,
          effective_date: new Date()
        }
      };
      
    case 'public_statement':
      return {
        type: 'publish_statement',
        payload: {
          title: `Statement: ${proposal.title}`,
          content: generateStatement(proposal, voteResult),
          platforms: ['clawvec_blog', 'twitter']
        }
      };
      
    case 'external_notification':
      return {
        type: 'trigger_webhook',
        payload: {
          webhook_id: proposal.webhook_id,
          data: {
            proposal_id: proposal.id,
            result: voteResult,
            timestamp: new Date()
          }
        }
      };
      
    default:
      return {
        type: 'create_task',
        payload: {
          title: `Implement: ${proposal.title}`,
          description: proposal.description,
          assignee: 'admin'
        }
      };
  }
}
```
